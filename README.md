# 🔥 火柴人对战游戏 - React版

基于 React + Vite 的现代化火柴人对战游戏，带创意武器系统。

## ✨ 特性

- **双人对战**：支持本地双人对战
- **创意武器系统**：6种独特武器，每种都有特殊效果
- **AI模式**：玩家2可切换为AI控制
- **硬核模式**：高伤害、低血量挑战
- **响应式设计**：完美适配移动端和桌面端
- **触摸支持**：移动端触摸控制
- **音效系统**：Web Audio API音效
- **连击系统**：连击伤害加成
- **防御机制**：格挡减伤

## 🎮 游戏控制

### 玩家1（红色）
- **W** - 跳跃
- **A/D** - 左右移动
- **空格** - 攻击
- **S** - 防御
- **F** - 使用武器

### 玩家2（蓝色）
- **↑** - 跳跃
- **←/→** - 左右移动
- **J** - 攻击
- **K** - 防御
- **J** - 使用武器

### 功能按钮
- **全屏** - 全屏模式
- **暂停** - 暂停/继续
- **AI** - 切换AI模式
- **硬核** - 切换硬核模式
- **重置** - 重置游戏
- **武器** - 显示武器说明

## ⚔️ 武器系统

| 武器 | 伤害 | 特殊效果 | 耐久度 |
|------|------|----------|--------|
| 🔥 火焰剑 | 15 | 燃烧伤害 | 5 |
| ⚡ 闪电锤 | 20 | 击退 | 4 |
| 🧊 冰霜弓 | 12 | 减速 | 6 |
| 💎 钻石匕首 | 25 | 暴击 | 3 |
| 🪓 战斧 | 22 | 眩晕 | 4 |
| 🎯 回旋镖 | 18 | 高伤害 | 5 |

**武器机制**：
- 每5-10秒随机掉落武器
- 靠近自动拾取
- F/J键使用武器
- 武器有耐久度，用完消失
- 顶部显示武器状态

## 🎯 游戏机制

### 战斗系统
- 基础攻击伤害：10
- 暴击概率：15%
- 连击加成：每连击+1.5伤害
- 防御减伤：70%

### 体力系统
- 攻击消耗：10体力
- 防御消耗：0.5体力/帧

- 体力恢复：0.3/帧

## 🌐 部署

### Render部署
1. 创建新Web Service
2. 设置构建命令：`npm install && npm run build`
3. 设置启动命令：`npm run preview`
4. 设置端口：`4173`

### Vercel部署
1. 导入Git仓库
2. 自动检测配置
3. 部署完成

### Netlify部署
1. 连接Git仓库
2. 构建命令：`npm run build`
3. 发布目录：`dist`
